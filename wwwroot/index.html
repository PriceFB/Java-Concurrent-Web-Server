Concurrent Web Server Demo

This page is served by the custom multithreaded server.

Available Endpoints

  • / - Static content (this page)
  • /echo.html - Instructions for the POST /echo endpoint
  • /slow.html - Triggers the /slow endpoint for concurrency demos
  • /slow - GET endpoint that waits 3 seconds
  • POST /echo - Echoes request bodies
  • /health - Lightweight health probe

Try a Quick Test

Use curl or a browser to open /slow in multiple tabs. Because each connection is processed in its own worker thread, the requests complete concurrently instead of sequentially.
